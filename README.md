## Prerequisites

- NodeJS >= 17

## Install

```bash
$ npm run install
```

## Usage

Run the website locally

```bash
$ npm run dev
```

Run automatic code formatting:

```bash
$ npm run lint
```

Build website
```bash
$ npm run build
```

The run on production

```bash
$ npm run start
```

## Settings

All configuration is in `next.config.js` and `next-i18next.config.js`.
